A mobile-first web application for golf tempo training. This app helps golfers maintain consistent 3:1 (Driver) or 2:1 (Approach) swing ratios using visual and audio cues.

## üåü Key Features
- **Precise Audio Engine**: Metronome (woodblock), Whistle, and Beep modes.
- **Visual Sync**: Real-time video playback synchronized with BPM and ratio.
- **PWA Support**: Can be installed on mobile home screens for a standalone app experience.
- **Interactive UI**: Neon-styled BPM display, 4-step rhythm dots, and touch-optimized controls.
- **Haptic Feedback**: Screen vibration (shake) on impact for better feel.
- **Milestone Celebration**: Fanfare and visual rewards for every 10 practice swings.

## üì± How to Use
1. **BPM Setting**: Adjust your target tempo using the slider or +/- buttons.
2. **Ratio Choice**:
   - **3:1**: 3-beat backswing, 1-beat downswing (Standard).
   - **2:1**: 2-beat backswing, 1-beat downswing (Aggressive/Short game).
3. **Sound**: Choose your preferred rhythm sound.
4. **Practice**: Tap **START** and follow the rhythm!
